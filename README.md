# PatientToMoney

【アプリケーションの概要】  
何か節約のために我慢しよう！そう思ってもどのくらい我慢したのかは意外とわからないですよね。  
そんな悩みを解決するアプリ、「我慢をお金に！」は自分が我慢した分のお金を可視化できます。  

【アプリケーションの機能一覧】  
1.匿名ログイン機能  
2.我慢項目投稿機能  
3.我慢項目一覧化機能  
4.我慢項目編集機能  
5.我慢項目削除機能  
6.アラート画面の出力機能  
７.我慢履歴表示機能 
  
【使用したライブラリ】  
Firebase Authentication  
Firebase FireStore
